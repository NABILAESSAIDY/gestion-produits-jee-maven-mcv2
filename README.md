# 🧩 Gestion des Produits - JEE (Hibernate : HBM vs Annotations)

Ce dépôt contient deux implémentations d’une application de gestion des produits en Java EE utilisant Hibernate, afin de comparer deux approches de mapping :

* 🔹 Configuration via fichiers XML (HBM)
* 🔹 Configuration via annotations Java

## 🎯 Objectif

L’objectif de ce projet est de comprendre et comparer deux méthodes de configuration d’Hibernate :

* Approche classique basée sur des fichiers XML (`.hbm.xml`)
* Approche moderne basée sur les annotations Java

Cette comparaison permet de mieux saisir les avantages, les limites et les cas d’utilisation de chaque méthode.

---

## ⚙️ Technologies utilisées

* Java
* JEE (Java Enterprise Edition)
* Hibernate ORM
* Maven

---

## 🔍 Comparaison des approches

| Critère           | HBM (XML) 🧾      | Annotations 🏷️       |
| ----------------- | ----------------- | --------------------- |
| Lisibilité        | Moyenne           | Élevée                |
| Maintenance       | Plus lourde       | Plus simple           |
| Flexibilité       | Élevée            | Moyenne               |
| Popularité        | Ancienne approche | Moderne (recommandée) |


---

## 💡 Remarques

* Les deux versions implémentent les mêmes fonctionnalités métier
* Seule la configuration Hibernate diffère
